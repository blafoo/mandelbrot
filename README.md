# Apfelmännchen aka Mandelbrot set

My own little application to visualize Apfelmännchen (aka the Mandelbrot set)

### Modules
##### Core 
Basic library with functionality about Complex numbers and Mandelbrot set (calculations) 
##### Vaadin 
Application and UI to visualize the Mandelbrot set

Enjoy!
